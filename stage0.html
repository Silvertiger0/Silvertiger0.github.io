Graffiti!?

By using tags, we can manipulate how our text behaves on our page.For example, we can make words BOLD by surrounding it with <b> text here </b> or ITALICIZE with <em> text there </em>. You can even go as far as to make a link to some amazing stuff!

In the box or outside?

Elements use two different types, inline and block. Tags that are for block elements create a box where you can have your text while inline is purely text. With block elements, you can have height and width which will give you more options for customization that we will learn about later.
